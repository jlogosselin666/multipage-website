Subtypes


Ever noticed the small numbers placed at the end of a value (either decimal, octal, binary or hexadecimal)?


These little numbers are called SUBTYPES.


A SUBTYPE is used to specify which sort of value we are working with.


Since we have 4 general types of values, there are therefore 4 sorts of subtypes:




  • subtype 10 to specify when a value is decimal
  • subtype 2 to specify when a value is binary
  • subtype 8 to specify when a value is octal
  • subtype 16 to specify when a value is hexadecimal.
