Decimal conversion



This important first part contains all the essential information and techniques to manually convert:



  • Decimal to Binary
  • Decimal to Octal
  • Decimal to Hexadecimal


Part 1: Decimal-to-Binary

To manually convert a decimal value to a binary expression, the method is quite simple.


Let's take the decimal value 75 (any decimal value will do).


The following steps show us how to convert this decimal value into a binary expression:


75 / 2 = 37

---> remainder of the division = 1

(because (37 x 2) + 1 = 75)


37 / 2 = 18

---> remainder of the division = 1


18 / 2 = 9

---> remainder of the division = 0


9 / 2 = 4

---> remainder of the division = 1


4 / 2 = 2

---> remainder of the division = 0


2 / 2 = 1

---> remainder of the division = 0


1 / 2 = 0

---> remainder of the division = 1

(because (0 x 2) + 1 = 1)


As soon as the division reaches 0 , we stop!


Let's read all the remainder values from top to bottom:


1 1 0 1 0 0 1


Now let's invert the order of these numbers, as so:


1 0 0 1 0 1 1


And voila!


We have the binary expression for the decimal value 75!

Part 2: Decimal-to-Octal

To manually convert a decimal value to an octal expression, the same technique shown above for converting to binary is used (this time dividing by 8 instead of 2).


Let's illustrate this with the decimal value 523:


523 / 8 = 65

---> remainder of the division = 3

(because (65 x 8) + 3 = 523)


65 / 8 = 8

---> remainder of the division = 1


8 / 8 = 1

---> remainder of the division = 0


1 / 8 = 0

---> remainder of the division = 1

(because (0 x 8) + 1 = 1)


As soon as the division reaches 0 , we stop!


Let's read all the remainder values from top to bottom:


3 1 0 1


Now let's invert the order of these numbers, as so:


1 0 1 3


And voila!


You have the octal expression for the decimal value 523!

Part 3: Decimal-to-Hexadecimal

To manually convert a decimal value to a hexadecimal expression, the techniques used are slightly different from the first two shown above.


Indeed, if binary expressions are restricted to 0s and 1s, and octal expressions do not allow 8s or 9s, hexadecimal expressions on the contrary have added characters, namely: all digits from 0 to 9, but also all letters from A to F (either in uppercase or lowercase).


The decimal value for each of these 15 characters is as follows:


  • character 016 = 010
  • character 116 = 110
  • character 216 = 210
  • character 316 = 310
  • character 416 = 410
  • character 516 = 510
  • character 616 = 610
  • character 716 = 710
  • character 816 = 810
  • character 916 = 910
  • character A16 = 1010
  • character B16 = 1110
  • character C16 = 1210
  • character D16 = 1310
  • character E16 = 1410
  • character F16 = 1510

Still unsure about those little subtype numbers added at the end of every value? Click here to know more!

Let's illustrate this with the decimal value 7562:


7562 / 16 = 472

---> remainder of the division = 10

(because (472 x 16) + 10 = 7562)

Hold on! We can see in the hexadecimal table that decimal 1010 is equivalent to A16!

---> therefore: remainder of the division = A


472 / 16 = 29

---> remainder of the division = 8


29 / 8 = 1

---> remainder of the division = 13

Hold on! We can see in the hexadecimal table that decimal 1310 is equivalent to D16!

---> therefore: remainder of the division = D


1 / 16 = 0

---> remainder of the division = 1

(because (0 x 16) + 1 = 1)


As soon as the division reaches 0 , we stop!


Let's read all the remainder values from top to bottom:


A 8 D 1


Now let's invert that line, as so:


1 D 8 A


And voila!


We have the hexadecimal expression for the decimal value 7562!
