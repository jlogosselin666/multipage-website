Octal conversion




Essentially, the same logic seen above with binary expressions applies to octal expressions, albeit with an added characteristic: the index again becomes the exponent (this time of 8), but the result is then multiplied with the number to which that index belongs!


'Nough said: an example will clarify things!


Say we have the octal expression:


7 1 2 0 3 6


With INDICES, the octal expression now looks like this:


75 14 23 02 31 60


Again, let us start with the leftmost digit:


75 14 23 02 31 60


First, we set the index 5 as exponent to base 8:

85 = 32,768

We then multiply this value by the number to which the "index/exponent" is attached:

32,768 x 7 = 229,376


We move on to the next digit:


75 14 23 02 31 60


We set the index 4 as exponent to base 8:

84 = 4,096

We then multiply this value by the number to which the "index/exponent" is attached:

4,096 x 1 = 4,096


We move on to the next digit:


75 14 23 02 31 60


Again, we set the index 3 as exponent to base 8:

83 = 512

We then multiply this value by the number to which the "index/exponent" is attached:

512 x 2 = 1,024


We move on to the next digit:


75 14 23 02 31 60


The index 2 is set as exponent to base 8:

82 = 64

We then multiply this value by the number to which the "index/exponent" is attached:

64 x 0 = 0


We move on to the next digit:


75 14 23 02 31 60


The index 1 is set as exponent to base 8:

81 = 8

We then multiply this value by the number to which the "index/exponent" is attached:

8 x 3 = 24


We move on to the last and final digit:


75 14 23 02 31 60


The index 0 is set as exponent to base 8:

80 = 1

We then multiply this value by the number to which the "index/exponent" is attached:

6 x 1 = 6

We now add all of the values found above:


229,376 + 4,096 + 1024 + 0 + 24 + 6 = 234,526


Recap time!


Converting an octal expression to a decimal value consists of the following steps:


  1. assign an index to every digit of the octal expression (starting with index 0 on the rightmost digit)
  2. use that index number as an exponent, this time of base 8
  3. multiply the result with the number to which is attached the "index/exponent"
  4. finally, add all the values found, which gives you the decimal value of the octal expression!

In this chapter, we only looked at how to convert an octal expression to a decimal.


To convert to the other two sorts (binary & hexadecimal), simply use the decimal result of your octal-to-decimal conversion and follow the techniques illustrated in the Decimal chapter:
